# Instructions

Before you begin backend development.

## If you need to do backend work, spin up the local backend service with

1. cd apps/backend
2. pnpm db:start
3. pnpm dev
4. pnpm db:stop - when done

This will run a node.js backend (port 3000) and postgres instance.

## If you need to do front work, spin up the local frontend service with

1. cd apps/frontend
2. pnpm dev

This will run a next.js app (port 3001)
